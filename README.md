# Emergency-Index-Pakistan-Rescue-Services-1122

**Emergency Index 1122** is an AI-powered system designed to automate the analysis and classification of emergency calls for Pakistan’s Rescue 1122 service. The system processes Urdu audio calls, converts them into text, translates them into English, and classifies them into categories such as **medical, fire, accident, or normal**.

## 🔧 Features
- 🎤 Urdu Speech-to-Text using Google Speech Recognition API  
- 🌐 Urdu to English Translation using Google Translate API  
- 🧠 NLP Pipeline (preprocessing, keyword extraction, feature engineering)  
- 🤖 Machine Learning Classification (Random Forest Model)  
- 💻 Interactive Streamlit Web Interface  
- 🗂️ Database storage for call logs, transcripts, and predictions  

## 🚀 Tech Stack
- Python, Streamlit  
- NLTK, SpaCy, TF-IDF  
- Random Forest (Scikit-learn)  
- Google APIs (Speech + Translate)  
- SQLite / MySQL  

## 📌 Workflow
1. Input Urdu audio call  
2. Convert speech → text  
3. Translate Urdu → English  
4. Apply NLP processing  
5. Classify emergency type  
6. Display result with response plan  

## 🎯 Outcome
- Automated emergency call triage  
- Improved response speed and accuracy  
- Scalable AI solution for public safety
